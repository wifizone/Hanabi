Содержимое *.in будет подаваться программе на вход через консоль.

Вывод программы должнен в точности совпадать с соответствующим файлом *.out

Файл *.txt можете использовать для изучения разных тонкостей и расхождений вашей реализации с реализацией жюри.

Поместите все файлы этой папки рядом с вашим exe-файлом и запускайте тестирование на конкретном входе такой командой:
test.bat solution.exe 1-1.in
